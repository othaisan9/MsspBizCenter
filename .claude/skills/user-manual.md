## 기능별 매뉴얼 목록

### EASM 핵심 기능
1. 대시보드 - 보안 현황 한눈에 보기
2. 자산 관리 - 외부 노출 자산 등록 및 관리
3. 위협 관리 - 탐지된 위협 분석 및 대응
4. 서비스 관리 - 포트/서비스 모니터링
5. 스캔 - 취약점 스캔 실행 및 결과 확인
6. 스케줄 - 정기 스캔 예약
7. Shadow IT - 미승인 자산 탐지 및 처리
8. 리포트 - PDF/Excel 보고서 생성

### 설정 기능
1. 테넌트 설정 - 조직 정보 관리
2. 사용자 관리 - 팀원 초대 및 권한 설정
3. 자산 태그 - 자산 분류 태그 관리
4. API 키 - 외부 연동 API 키 설정
