# MsspBizCenter 목업

간단한 HTML/CSS 목업 페이지입니다.

## 사용법

브라우저에서 직접 열기:

```bash
# 메인 대시보드
open mockup/index.html

# 또는 Live Server로 실행 (VSCode)
# mockup/index.html 우클릭 > "Open with Live Server"
```

## 페이지 구성

### 목록 페이지
- `index.html` - 메인 대시보드
- `tasks.html` - 업무 일지 (칸반 보드)
- `meetings.html` - 회의록 목록
- `contracts.html` - 계약 관리 목록

### 입력 페이지
- `task-new.html` - 새 Task 등록
- `meeting-new.html` - 새 회의록 작성
- `contract-new.html` - 새 계약 등록

## 기술 스택

- **HTML5**
- **Tailwind CSS CDN** (스타일링)
- **정적 데이터** (하드코딩)

## 특징

- ✅ Backend 불필요

- ✅ 미시시피 스타일 유지 (카드 레이아웃)
